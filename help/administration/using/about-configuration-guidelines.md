# 設定指南 {#about-configuration-guidelines}

## 相容瀏覽器{#compatible-browsers}

Adobe Campaign Standard與大部分的電腦瀏覽器以及平板電腦的主要作業系統相容。

Adobe Campaign Standard僅支援橫向作為平板電腦上的顯示模式。

以下是與Adobe Campaign Standard相容的瀏覽器清單：

**電腦：**

* Chrome （最新版本）
* Firefox （最新版本）
* Safari （最新版本）
* Microsoft Edge （最新版本）

**平板電腦：**

* iOS （第7版和更新版本）
* Android （4.4版和更新版本）
