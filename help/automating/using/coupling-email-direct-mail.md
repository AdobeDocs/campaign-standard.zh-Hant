# 串聯電子郵件與直接郵件傳遞 {#coupling-email-direct-mail}

身為行銷人員，您可能想要透過直效郵件傳送型錄。在紙本型錄中，某些頁面會使用促銷代碼和連結，透過網站購買產品可享有 10% 的折扣。

離線和線上的結合十分有趣，因為有些客戶寧願線上訂購，但偏好在書面上檢視產品。
