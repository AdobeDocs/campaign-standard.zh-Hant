# Workflow data and processes{#workflow-data-and-processes}

透過Adobe Campaign，您可以匯入、轉換、篩選、管理和匯出資料。這些作業可進行，以設計和自動化您的業務流程。

* 更新資料庫：安裝封裝或上傳檔案的資料。
* 自動化程序：透過工作流程，您可以設計自己的程序來轉換資料、更新資料庫、傳送訊息等等。
* 匯出資料：匯出套件或資料集。
