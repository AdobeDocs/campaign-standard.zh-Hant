# 工作流程資料和程序{#workflow-data-and-processes}

有了Adobe Campaign，您就可以匯入、轉換、篩選、管理和匯出資料。 這些操作可用於設計和自動化業務流程。

* 更新資料庫：安裝套件或從檔案上傳資料。
* 自動化流程：透過工作流程，您可以設計自己的流程來轉換資料、更新資料庫、傳送訊息等。
* 匯出資料：將套件或資料集匯出至個別檔案。
