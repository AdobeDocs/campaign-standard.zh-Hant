# 著陸頁面內容編輯器介面{#landing-page-content-editor-interface}

著陸頁面內容編輯器可讓您輕鬆定義、修改及個人化Adobe Campaign中的內容。 若要存取它，請按一 **[!UICONTROL Content]** 下著陸頁面控制面板中的區塊。

內容編輯器會組織成三個不同的區段。 這些區段可讓您檢視和編輯內容。

![](assets/des_lp_content_8.png)

1. 畫面 **左側的** 「浮動視窗」可讓您修改連結至選取區塊的一般選項。 可修改的選項包括：背景顏色、邊框、文字對齊、可見性條件等。 請參 [閱插入個人化欄位](../../designing/using/personalization.md#inserting-a-personalization-field)。
1. 動 **作列** ，包含頁面的一般選項。 您可以選取範本並變更顯示模式。 請參 [閱著陸頁面編輯器動作列](../../channels/using/landing-page-content-editor-interface.md#landing-page-editor-action-bar)。
1. 主編 **輯區** ，可讓您使用內容相關工具列直接與內容互動：將連結插入影像、變更字型、刪除欄位等。 請參 [閱著陸頁面編輯器工具列](../../channels/using/landing-page-content-editor-interface.md#landing-page-editor-toolbar)。

## 著陸頁面編輯器工具列 {#landing-page-editor-toolbar}

工具列是編輯器 **介面的內容元素** ，根據選取的區域提供各種功能。 它包含可讓您變更文字樣式的動作按鈕和按鈕。 所執行的修改一律適用於選取的區域。 選擇塊後，可以刪除或複製它。 在選取區塊內的文字後，您可以將它轉換為連結或加粗。

![](assets/delivery_content_17.png)

>[!CAUTION]
>
>某些工具列功能可讓您設定HTML內容的格式。 不過，如果頁面包含CSS樣式表，則樣 **式表中的說明** ，可能會優先於透過工具列 **指定的指示** 。
