# About extending the API{#about-extending-the-api}

此使用案例詳細說明公開Profiles API中自訂資源描述檔的延伸欄位位置。例如，透過帳單系統產生的促銷代碼立即更新客戶的個人檔案。

In order to publish the API, you need to have the **[!UICONTROL Datamodel]** role.

* 擴充描述檔自訂資源
* 發佈擴充功能
* 驗證API中是否有延伸欄位
