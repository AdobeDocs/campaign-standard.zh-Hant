# Complaints{#complaints}

**[!UICONTROL Complaints]** 報表識別收到最多宣告訊息的傳送。

![](assets/delivery_reports_complaints.png)

**Flep** 表格按照收件者網域排序，顯示已宣告為電子郵件或垃圾郵件的收件者數目。表格的結果也可用於圖表和摘要數字。

**「傳送與黑名單」比率** 表列出將電子郵件宣告為垃圾郵件或垃圾郵件的收件者人數。表格會按照傳送順序排序。
