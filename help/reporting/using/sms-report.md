# 簡訊報告{#sms-report}

的 **簡訊** 報告提供了有關SMS交付的詳細資訊，如交付和彈跳率。

![](assets/dynamic_report_sms.png)

的 **簡訊摘要** 表、圖形和摘要編號包含已發送的SMS遞送的可用資料。

* **已處理/已發送**:發送的SMS數。
* **已交付**:發送的SMS數。
* **邊界+錯誤**:無法傳遞的消息數。
