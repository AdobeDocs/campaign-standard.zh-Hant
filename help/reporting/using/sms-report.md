# 簡訊報告{#sms-report}

此 **簡訊** 報表提供SMS傳送的詳細資訊，例如傳送和反彈率。

![](assets/dynamic_report_sms.png)

此 **簡訊摘要** 表格、圖表和摘要數字包含已傳送之SMS傳送的可用資料。

* **已處理/已傳送**:已傳送的簡訊數。
* **傳遞**:傳送的簡訊數量。
* **跳出數+錯誤**:無法傳遞的郵件數。
