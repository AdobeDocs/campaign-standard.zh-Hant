# SMS報告{#sms-report}

**SMS** 報告提供SMS傳送的詳細資訊，例如傳送和反彈率。

![](assets/dynamic_report_sms.png)

**SMS摘要** 表格、圖形和摘要數字包含已傳送之SMS傳送的可用資料。

* **已處理/傳送**：傳送的SMS數目。
* **遞送**：傳送的SMS數目。
* **彈回數+錯誤**：無法傳送的訊息數。
