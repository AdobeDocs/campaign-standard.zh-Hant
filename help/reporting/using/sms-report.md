# SMS 報告{#sms-report}

**SMS**&#x200B;報表提供有關SMS傳送的詳細資訊，例如傳送率和反彈率。

![](assets/dynamic_report_sms.png)

**SMS摘要**&#x200B;表格、圖形和摘要編號包含已發送的SMS傳送的可用資料。

* **已處理／已傳送**:傳送的簡訊數。
* **交付**:傳送的簡訊數。
* **彈回數+錯誤**:無法傳送的訊息數。
