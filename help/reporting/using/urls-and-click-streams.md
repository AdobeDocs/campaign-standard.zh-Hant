# URL 和點擊流量{#urls-and-click-streams}

**URL和點按資料流**&#x200B;會顯示傳遞期間最多點按的URL，或若是連結至行銷活動或方案的多個傳遞。

![](assets/delivery_reports_8.png)

每個表格都由摘要數字和圖表表示。 您可以變更詳細資訊在其各自視覺效果設定中的顯示方式。

**最常造訪的連結**&#x200B;表格包含每個傳遞的收件者行為可用資料，例如：

* **點按**：內容在傳遞中被點按的次數。
* **不重複點按**：點按傳遞中內容的收件者人數。
