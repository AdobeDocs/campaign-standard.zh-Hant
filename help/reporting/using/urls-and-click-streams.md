# URL 和點擊流量{#urls-and-click-streams}

的 **URL並按一下流** 顯示在交貨期間按一下次數最多的URL，或者，如果連結到市場活動或程式，則顯示多個交貨。

![](assets/delivery_reports_8.png)

每個表都由摘要數字和圖表表示。 您可以更改詳細資訊在其各自的可視化設定中的顯示方式。

的 **訪問次數最多的連結** 表包含每個傳遞的收件人行為的可用資料，如：

* **按一下**:在傳遞中按一下內容的次數。
* **按一下**:按一下傳遞中內容的收件人數。
