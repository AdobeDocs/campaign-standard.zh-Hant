## 版本 26.2 {#26.2}

### 安全性修正 {#26-2-security}

* 此版本提供安全性修正。
* 此版本隨附下列安全性升級：Debian 13 和 PostgreSQL 17。

### 其他修正 {#26-2-fixes}

* 改進的動態報告具有更好的穩健性並減少了資料不符項目。 (CAMP-58964)
* 改善了 IMS 登入機制以提高可靠性。 (CAMP-59418)
* 改善了 AEM 內容整合，針對分頁請求提供更好的 URL 處理功能。 (CAMP-59540)
* 在「選項」畫面中新增密碼類型輸入支援，並具備自動雜湊功能。 (CAMP-60206)
* 修正了臨時工作流程使用錯誤序列 ID 的問題。 (CAMP-59824)
* 修正了擷取 AEM 行銷活動內容時，可能導致錯誤的問題。 (CAMP-59926)
