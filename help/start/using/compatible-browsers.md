# Compatible browsers{#compatible-browsers}

Adobe Campaign Standard與大部分電腦的瀏覽器相容，以及平板電腦的主要作業系統相容。

Adobe Campaign Standard只支援平板電腦上的橫向顯示模式。

以下是與Adobe Campaign Standard相容的瀏覽器清單：

**電腦：**

* Chrome(最新版本)
* Firefox(最新版本)
* Safari(最新版本)
* Microsoft Edge(最新版本)

**平板電腦：**

* iOS(版本及更新版本)
* Android(4.4版及更新版本)
