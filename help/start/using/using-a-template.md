>[!NOTE]
>
>根據範本建立行銷活動通常是由標準使用者類型設定檔所進行。

1. 建立新的行銷活動。

   ![](assets/template_5.png)

   ![](assets/template_6.png)

   行銷活動已預先設定為範本中定義的參數。
